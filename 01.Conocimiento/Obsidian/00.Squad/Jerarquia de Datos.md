# Jerarquia de Datos

## Niveles

| Nivel | Datos | Acceso |
|-------|-------|--------| 
| Publico | Agenda general, preferencias | Todos |
| Interno | Finanzas agregadas, historial resumido | Agente + Sandra |
| Confidencial | Detalles medicos, financieros detallados | Solo agente |

## Reglas

- Cada agente accede solo a lo necesario para su funcion
- Sandra tiene vision transversal para coordinacion
- Datos medicos sensibles: solo el agente medico
- Datos financieros detallados: solo el agente de finanzas
