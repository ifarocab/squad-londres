# SELF_IMPROVEMENT_SHARED.md

## Objetivo

Mejorar el sistema del squad sin perder coherencia ni estabilidad.

## Governance by Lucia

Lucía valida mejoras que afectan a múltiples agentes, detecta incoherencias, consolida patrones útiles y evita degradación del sistema.

## Regla

- mejora local → no entra aquí
- mejora de squad → registrar en `00.Gobierno/00.Queue/governance-queue.md`
- si no está en governance-queue, no existe como mejora de squad

## Qué revisar

- calidad de delegaciones
- calidad de handoffs
- coste de coordinación
- valor real de subagentes
- patrones de retrabajo o ruido
- incidentes de contexto repetidos
- inflación progresiva por automejora

## Context budget rule

Si una instrucción no cambia decisiones, sobra contexto.

Priorizar:
- reglas operativas
- criterios de decisión
- referencias bajo demanda para detalle
