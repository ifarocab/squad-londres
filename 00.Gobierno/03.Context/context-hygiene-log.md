# context-hygiene-log.md

Registrar solo incidencias relevantes de contexto:

- truncados de bootstrap o contexto inyectado
- timeouts donde el contexto sea causa probable
- duplicidad persistente entre archivos base
- compactaciones o podas aplicadas
- propuestas transversales para reducir contexto

Formato sugerido:

## [YYYY-MM-DD] Incidencia
- agente:
- señal:
- causa probable:
- acción tomada:
- resultado:
- escalado: no | sí
