# 00.Gobierno

Carpeta única de gobierno del squad.

## Estructura
- `00.Queue/`: propuestas pendientes de mejora o gobierno.
- `01.Decisions/`: decisiones aprobadas o rechazadas.
- `02.Knowledge/`: conocimiento operativo compartido del squad.
- `03.Context/`: incidencias y seguimiento de higiene de contexto.

## Regla
Si una mejora de squad no está en `00.Queue/governance-queue.md`, no existe como mejora de squad.
