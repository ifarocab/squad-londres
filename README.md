## Uso

### Crear issue

1. Ir a Issues > New Issue
2. Seleccionar plantilla segun tipo:
   - Analisis Medico → Para Luis
   - Seguimiento Financiero → Para Miguel
   - Planificacion Nutricional → Para Nerea
   - Coordinacion de Squad → Para Sandra
3. Completar informacion y asignar labels

### Flujo de trabajo

``nIgnacio → Sandra (preferente)
              |
              |-- Tema medico claro → Luis
              |-- Tema financiero claro → Miguel
              |-- Tema nutricion claro → Nerea
              |-- Resto → Sandra gestiona
``n
